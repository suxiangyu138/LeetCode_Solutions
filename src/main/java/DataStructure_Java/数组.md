# 快速吃透「数组」，零基础秒懂

## 一、数组核心定义
**数组**：**相同数据类型**的一组数据，在内存中**连续存放**的有序集合。
核心特点：
1. 类型统一：只能存同一种数据（int 就只能存整数）
2. 长度固定：创建后不能扩容/缩容
3. 下标索引：从 **0** 开始编号，快速取值

---

## 二、关键基础概念
1. **下标/索引**
   数组第 1 个元素：下标 `0`
   数组第 2 个元素：下标 `1`
   以此类推，最后一个元素下标 = 数组长度 - 1

2. **数组长度**
   固定属性：`数组名.length`

3. **内存特点**
   连续空间，查询极快（通过下标一步定位）；增删元素慢（需要移动数据）

---

## 三、Java 数组三种创建方式
### 方式1：静态初始化（直接赋值）
```java
// 完整写法
int[] arr1 = new int[]{10,20,30,40};
// 简写（最常用）
int[] arr2 = {1,2,3,4};
```

### 方式2：动态初始化（只指定长度，默认补初始值）
```java
// 创建长度为5的int数组，默认值：0
int[] arr = new int[5];
```
默认初始值规则：
- 整数：`0`
- 小数：`0.0`
- 布尔：`false`
- 引用类型：`null`

### 方式3：先声明、后赋值
```java
int[] arr;
arr = new int[]{5,6,7};
```

---

## 四、数组基础操作（增删改查核心）
### 1. 查：通过下标获取元素
```java
int[] arr = {11,22,33};
// 取下标0元素
System.out.println(arr[0]); 
```

### 2. 改：通过下标修改元素
```java
arr[1] = 99; 
```

### 3. 遍历（全部取出）
#### 普通 for 循环（可改元素）
```java
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

#### 增强 for 循环（只读取，简洁）
```java
for (int num : arr) {
    System.out.println(num);
}
```

---

## 五、两大常见异常
1. **数组下标越界异常**
   `ArrayIndexOutOfBoundsException`
   原因：访问了不存在的下标（例如长度3，却取 `arr[5]`）

2. **空指针异常**
   `NullPointerException`
   原因：数组只是声明，没有 `new` 初始化

---

## 六、数组优缺点总结
### 优点
- 下标随机访问速度极快
- 结构简单，占用内存紧凑

### 缺点
- 长度固定，灵活性差
- 中间插入/删除元素效率低
- 只能存单一类型数据

---

# 数组入门必刷 5 题（极简版，直接上手）
语言：Java，只考数组基础：遍历、取值、改值、长度、下标

## 题目1：数组求和
需求：给定数组，计算所有元素总和
```java
public class Main {
    public static void main(String[] args) {
        int[] arr = {1,2,3,4,5};
        int sum = 0;
        for(int i = 0; i < arr.length; i++){
            sum += arr[i];
        }
        System.out.println(sum);
    }
}
```

## 题目2：找数组最大值
需求：遍历数组，找出最大数字
```java
public class Main {
    public static void main(String[] args) {
        int[] arr = {12,45,7,99,23};
        int max = arr[0];
        for(int i = 1; i < arr.length; i++){
            if(arr[i] > max){
                max = arr[i];
            }
        }
        System.out.println(max);
    }
}
```

## 题目3：数组元素反转
需求：`[1,2,3,4]` → `[4,3,2,1]`
```java
public class Main {
    public static void main(String[] args) {
        int[] arr = {1,2,3,4};
        for(int i = 0; i < arr.length / 2; i++){
            int temp = arr[i];
            arr[i] = arr[arr.length - 1 - i];
            arr[arr.length - 1 - i] = temp;
        }
        for(int num : arr){
            System.out.print(num+" ");
        }
    }
}
```

## 题目4：元素查找（判断是否存在）
需求：输入目标数，判断是否在数组中
```java
public class Main {
    public static void main(String[] args) {
        int[] arr = {10,20,30,40};
        int target = 30;
        boolean flag = false;
        for(int i = 0; i < arr.length; i++){
            if(arr[i] == target){
                flag = true;
                break;
            }
        }
        System.out.println(flag ? "存在" : "不存在");
    }
}
```

## 题目5：统计偶数个数
需求：遍历数组，数出有几个偶数
```java
public class Main {
    public static void main(String[] args) {
        int[] arr = {2,5,8,11,14,17};
        int count = 0;
        for(int i = 0; i < arr.length; i++){
            if(arr[i] % 2 == 0){
                count++;
            }
        }
        System.out.println(count);
    }
}
```

我给你写**最简单、最标准、能直接运行**的 Java 数组求和程序，包含**最基础版**和**通用方法版**，你直接复制就能用。

# 1. 最基础版（新手必看）
```java
public class ArraySum {
    public static void main(String[] args) {
        // 1. 定义一个数组
        int[] arr = {1, 2, 3, 4, 5};

        // 2. 定义变量存总和
        int sum = 0;

        // 3. 遍历数组，累加
        for (int i = 0; i < arr.length; i++) {
            sum = sum + arr[i];  // 把每个元素加到 sum 里
        }

        // 4. 输出结果
        System.out.println("数组总和：" + sum);
    }
}
```

运行结果：
```
数组总和：15
```

# 2. 通用方法版（更规范、可复用）
```java
public class ArraySum {
    public static void main(String[] args) {
        int[] arr = {10, 20, 30, 40};
        int result = getSum(arr);  // 调用求和方法
        System.out.println("数组总和：" + result);
    }

    // 专门用来求数组和的方法
    public static int getSum(int[] arr) {
        int sum = 0;
        for (int num : arr) {  // 增强for循环，更简洁
            sum += num;
        }
        return sum;
    }
}
```

运行结果：
```
数组总和：100
```

# 核心逻辑（一定要懂）
1. 定义数组
2. 定义一个变量 `sum = 0` 用来存结果
3. **遍历数组**，把每一个元素**加到 sum 里**
4. 输出 sum

---

### 总结
1. 数组求和 = **遍历 + 累加**
2. 两种写法：**普通 for 循环**、**增强 for 循环**
3. 代码简单、面试笔试必考基础题
