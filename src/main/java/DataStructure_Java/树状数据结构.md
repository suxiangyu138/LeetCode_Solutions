# 极速吃透：树 数据结构 + 核心算法
## 一、树是什么（一句话）
**树是非线性数据结构**，元素一对多，层级关系。
和线性结构（数组/链表/栈/队列）对立。

---
## 二、基础术语（必背，极简）
- **根节点**：最顶层，唯一，无父节点
- **父/子节点**：上层为父，下层为子
- **叶子节点**：没有子节点
- **深度**：当前节点到根的层数
- **高度**：当前节点到最远叶子的层数
- **度**：节点拥有的子节点个数

---
## 三、最常用：二叉树
### 1. 定义
每个节点**最多2个子节点**：左孩子、右孩子。

### 2. 二叉树分类
1. **普通二叉树**：无限制
2. **满二叉树**：每一层全部长满
3. **完全二叉树**：最后一层从左往右填
4. **二叉搜索树 BST**
   左子树所有值 < 根 < 右子树所有值
   ✅ 高效查找、插入、删除
5. **平衡二叉树(AVL/红黑树)**：防止BST退化成链表

---
## 四、二叉树节点结构（Java 标准模板）
```java
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode(int x) { val = x; }
}
```

---
## 五、二叉树核心：四种遍历（必考）
### 记忆口诀
- 前序：**根 → 左 → 右**
- 中序：**左 → 根 → 右**（BST中序=有序数组）
- 后序：**左 → 右 → 根**
- 层序：从上到下、从左到右（借助队列）

### 递归代码（最简）
```java
// 前序
void preOrder(TreeNode root){
    if(root == null) return;
    System.out.print(root.val);
    preOrder(root.left);
    preOrder(root.right);
}

// 中序
void inOrder(TreeNode root){
    if(root == null) return;
    inOrder(root.left);
    System.out.print(root.val);
    inOrder(root.right);
}

// 后序
void postOrder(TreeNode root){
    if(root == null) return;
    postOrder(root.left);
    postOrder(root.right);
    System.out.print(root.val);
}
```

### 层序遍历（队列实现）
```java
import java.util.LinkedList;
import java.util.Queue;

void levelOrder(TreeNode root){
    if(root == null) return;
    Queue<TreeNode> q = new LinkedList<>();
    q.offer(root);
    while(!q.isEmpty()){
        TreeNode cur = q.poll();
        System.out.print(cur.val);
        if(cur.left != null) q.offer(cur.left);
        if(cur.right != null) q.offer(cur.right);
    }
}
```

## 六、树 高频经典算法
1. **递归类（核心思想：分治）**
    - 求树的最大深度
    - 求节点个数
    - 求叶子节点个数
    - 翻转二叉树（左右互换）

2. **二叉搜索树 BST 算法**
    - 查找目标值
    - 插入节点
    - 删除节点
    - 中序遍历得到升序序列

3. **路径问题**
    - 二叉树的直径
    - 根到叶子路径和

4. **迭代遍历**
   栈模拟递归，不用递归实现前/中/后序

---
## 七、树  vs  线性结构
- 数组/链表：一对一
- 树：**一对多**
- 栈、队列常用来：迭代遍历树、层序遍历

---
## 八、极简例题：求二叉树最大深度
```java
public int maxDepth(TreeNode root) {
    if(root == null) return 0;
    int left = maxDepth(root.left);
    int right = maxDepth(root.right);
    return Math.max(left, right) + 1;
}
```

---
## 九、总结速记
1. 树 = 层级、一对多
2. 二叉树每个节点最多两个孩子
3. 三大递归遍历：前、中、后
4. 层序遍历必用队列
5. BST 左小右大，中序有序
6. 树算法核心：**递归分治**

# 常用树相关算法（分类+核心思路+Java极简模板）

- **前序遍历**：根 → 左 → 右
- **中序遍历**：左 → 根 → 右

## 五、刷题高频模板（直接背）
### 1. 遍历递归通用模板
```java
class TreeNode{int val;TreeNode left,right;}

void dfs(TreeNode root){
    if(root == null) return;
    // 前序：写这里
    dfs(root.left);
    // 中序：写这里
    dfs(root.right);
    // 后序：写这里
}
```

### 2. 层序遍历（队列）
```java
import java.util.Queue;
import java.util.LinkedList;
void bfs(TreeNode root){
    if(root == null) return;
    Queue<TreeNode> q = new LinkedList<>();
    q.offer(root);
    while(!q.isEmpty()){
        TreeNode cur = q.poll();
        if(cur.left != null) q.offer(cur.left);
        if(cur.right != null) q.offer(cur.right);
    }
}
```

### 3. 树最大深度
```java
public int maxDepth(TreeNode root){
    if(root == null) return 0;
    return Math.max(maxDepth(root.left),maxDepth(root.right)) + 1;
}
```
