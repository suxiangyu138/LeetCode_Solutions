# 快速学会 图 数据结构 + 核心概念 + Java完整代码实现
## 一、图 核心概念（极简背诵）
### 1. 定义
**图是非线性结构**，由 **顶点(V)** 和 **边(E)** 组成，多对多关系。
- 顶点：节点
- 边：节点之间的连线

### 2. 分类
1. **有向图 / 无向图**
    - 无向：边无方向，双向连通
    - 有向：边有方向，单向通行
2. **带权图 / 无权图**
    - 带权：边有数值（距离、代价）
3. **连通图**：任意两点可达
4. **环图 / 无环图**

### 3. 基础术语
- 度：一个顶点连接的边数
- 入度/出度（有向图）
- 邻接：两点直接相连
- 路径、环、连通分量

---

## 二、图两种存储结构（必考）
### 1. 邻接矩阵
- 用**二维数组**存储
- `graph[i][j]`：i 到 j 是否连通/权重
- 优点：查询快、实现简单
- 缺点：稀疏图极度浪费空间

### 2. 邻接表（工程/刷题常用）
- 用**数组 + 链表/集合**
- 每个顶点维护一个列表，存相连节点
- 优点：节省空间、适合稀疏图
- 缺点：查询两点是否相连较慢

---

## 三、图两大核心遍历算法
1. **DFS 深度优先搜索**
   一路走到底，走不通再回溯，靠**递归/栈**
2. **BFS 广度优先搜索**
   一层一层扩散，靠**队列**

## 四、Java 完整代码实现
### 1. 邻接矩阵 实现 + DFS/BFS
```java
public class MatrixGraph {
    private int[][] graph;
    private int vertexNum;
    private boolean[] visited;

    public MatrixGraph(int n) {
        vertexNum = n;
        graph = new int[n][n];
        visited = new boolean[n];
    }

    // 添加无向边
    public void addEdge(int a, int b) {
        graph[a][b] = 1;
        graph[b][a] = 1;
    }

    // DFS
    public void dfs(int v) {
        visited[v] = true;
        System.out.print(v + " ");
        for (int i = 0; i < vertexNum; i++) {
            if (graph[v][i] == 1 && !visited[i]) {
                dfs(i);
            }
        }
    }

    // BFS
    public void bfs(int v) {
        boolean[] vis = new boolean[vertexNum];
        java.util.Queue<Integer> queue = new java.util.LinkedList<>();
        queue.offer(v);
        vis[v] = true;
        while (!queue.isEmpty()) {
            int cur = queue.poll();
            System.out.print(cur + " ");
            for (int i = 0; i < vertexNum; i++) {
                if (graph[cur][i] == 1 && !vis[i]) {
                    vis[i] = true;
                    queue.offer(i);
                }
            }
        }
    }

    public static void main(String[] args) {
        MatrixGraph g = new MatrixGraph(5);
        g.addEdge(0,1);
        g.addEdge(0,2);
        g.addEdge(1,3);
        g.addEdge(2,4);

        System.out.println("邻接矩阵 DFS：");
        g.dfs(0);
        System.out.println("\n邻接矩阵 BFS：");
        g.bfs(0);
    }
}
```

### 2. 邻接表 实现 + DFS/BFS（工作、刷题主流）
```java
import java.util.ArrayList;
import java.util.LinkedList;
import java.util.Queue;

public class ListGraph {
    private ArrayList<ArrayList<Integer>> adj;
    private int vertexNum;
    private boolean[] visited;

    public ListGraph(int n) {
        vertexNum = n;
        adj = new ArrayList<>();
        for (int i = 0; i < n; i++) {
            adj.add(new ArrayList<>());
        }
        visited = new boolean[n];
    }

    // 添加无向边
    public void addEdge(int a, int b) {
        adj.get(a).add(b);
        adj.get(b).add(a);
    }

    // DFS
    public void dfs(int v) {
        visited[v] = true;
        System.out.print(v + " ");
        for (int next : adj.get(v)) {
            if (!visited[next]) {
                dfs(next);
            }
        }
    }

    // BFS
    public void bfs(int v) {
        boolean[] vis = new boolean[vertexNum];
        Queue<Integer> queue = new LinkedList<>();
        queue.offer(v);
        vis[v] = true;
        while (!queue.isEmpty()) {
            int cur = queue.poll();
            System.out.print(cur + " ");
            for (int next : adj.get(cur)) {
                if (!vis[next]) {
                    vis[next] = true;
                    queue.offer(next);
                }
            }
        }
    }

    public static void main(String[] args) {
        ListGraph g = new ListGraph(5);
        g.addEdge(0,1);
        g.addEdge(0,2);
        g.addEdge(1,3);
        g.addEdge(2,4);

        System.out.println("\n\n邻接表 DFS：");
        g.dfs(0);
        System.out.println("\n邻接表 BFS：");
        g.bfs(0);
    }
}
```

## 五、图 高频算法清单
1. **遍历**
   DFS 深度优先、BFS 广度优先
2. **最短路径**
    - Dijkstra：单源最短路径（无负权）
    - Floyd：多源最短路径
    - Bellman-Ford：可处理负权
3. **最小生成树**
    - Prim 算法
    - Kruskal 算法
4. **拓扑排序**
   有向无环图、任务排程
5. **连通性**
   并查集（Union-Find）判断连通分量

## 六、快速总结
1. 图 = **多对多**，树是特殊的图
2. 存储：
    - 稠密图 → 邻接矩阵
    - 稀疏图 → 邻接表
3. 遍历：
    - DFS：递归/栈
    - BFS：队列
4. 刷题优先掌握：**邻接表 + DFS + BFS + 并查集**
