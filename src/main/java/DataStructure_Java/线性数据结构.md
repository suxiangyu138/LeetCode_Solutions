# 快速吃透：线性数据结构（极简易懂，全程对比记忆）

## 一、什么是线性结构
**核心定义**：
元素之间是**一对一**前后相连的关系，数据排成一条「直线」，只有**唯一前驱、唯一后继**。

五大经典线性结构：
**数组、链表、栈、队列、字符串**

---

## 二、逐个速学 + 核心特点 + 底层逻辑
### 1. 数组 Array
- 存储：内存**连续**
- 访问：下标随机访问超快
- 缺点：长度固定、中间增删慢
- 核心操作：查改快，增删慢

### 2. 链表 Linked List
- 存储：内存**不连续**，每个节点存「数据 + 下一个节点地址」
- 分类：单向链表、双向链表、循环链表
- 优点：动态扩容、任意位置增删快
- 缺点：只能从头遍历、查找慢

### 3. 栈 Stack
- 规则：**后进先出 LIFO**
- 比喻：子弹弹夹、叠盘子
- 核心操作：
  - 入栈 push
  - 出栈 pop
- 应用：括号匹配、函数调用、表达式运算、回溯算法

### 4. 队列 Queue
- 规则：**先进先出 FIFO**
- 比喻：排队买票
- 分类：
  - 普通队列：一端进、一端出
  - 循环队列：解决假溢出
  - 双端队列 Deque：两端都可进出
- 应用：消息排队、任务调度、广度优先搜索 BFS

### 5. 字符串 String
- 本质：**字符数组**
- 线性有序，属于特殊线性结构
- 重点：不可变、字符遍历、子串、拼接

## 三、线性结构 核心对比（必背）

| 结构     | 内存分布 | 查找效率      | 增删效率      | 核心规则/特点                |
|:-------|:-----|:----------|:----------|:-----------------------|
| **数组** | 连续   | $O(1)$ 极快 | $O(n)$ 慢  | 支持随机访问，长度固定            |
| **链表** | 分散   | $O(n)$ 慢  | $O(1)$ 快* | 只能顺序遍历，动态扩容            |
| **栈**  | 不限   | 不支持随机查找   | $O(1)$    | **后进先出 (LIFO)**，仅操作栈顶  |
| **队列** | 不限   | 不支持随机查找   | $O(1)$    | **先进先出 (FIFO)**，队尾进队头出 |

> *注：链表的增删快是指已知节点位置时修改指针的操作；若需先查找则整体为 $O(n)$。

---

## 四、线性结构通用 3 大基础操作
1. **遍历**
从头到尾逐个访问元素，所有线性结构都必须掌握。
2. **增删**
数组靠移位，链表改引用，栈/队列只操作边界。
3. **查找**
数组下标查找，链表逐个遍历。

## 五、Java 极简代码示例（快速上手）
### 1. 栈
```java
import java.util.Stack;
Stack<Integer> stack = new Stack<>();
stack.push(1);
stack.push(2);
stack.pop();
```

### 2. 队列
```java
import java.util.Queue;
import java.util.LinkedList;
Queue<Integer> queue = new LinkedList<>();
queue.offer(1);
queue.poll();
```

### 3. 单向链表节点结构
```java
class Node {
    int val;
    Node next;
}
```

## 六、线性结构 常见算法
1. 数组：双指针、前缀和、二分、排序
2. 链表：反转链表、快慢指针、环形判断、合并链表
3. 栈：括号匹配、单调栈
4. 队列：BFS、滑动队列

---

## 七、一句话终极总结
1. **数组**：连续内存，查快改快
2. **链表**：链式存储，增删自由
3. **栈**：后进先出
4. **队列**：先进先出
5. **字符串**：字符形式的线性数组
