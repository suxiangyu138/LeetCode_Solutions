# Java 排序算法速通（最核心+最常用）
我直接带你学**面试/开发必考的 6 大排序**，用最简单的 Java 代码 + 一句话原理，**看完就能写、能讲、能做题**。

先给你一张总览表：

| 算法   | 时间复杂度(平均)  | 空间       | 稳定性 | 核心思想       |
|------|------------|----------|-----|------------|
| 冒泡排序 | O(n²)      | O(1)     | 稳定  | 两两交换，大的往后冒 |
| 选择排序 | O(n²)      | O(1)     | 不稳定 | 选最小/大，放到前面 |

| 归并排序 | O(n log n) | O(n)     | 稳定  | 分治：先拆分，再合并 |

## 1. 冒泡排序（最简单）
**一句话**：相邻两个数比较，大的往后换，每轮冒出一个最大值。
```java
public static void bubbleSort(int[] arr) {
    int n = arr.length;
    // 外层：控制轮数
    for (int i = 0; i < n - 1; i++) {
        // 内层：两两比较
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                // 交换
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}
```

## 2. 选择排序
**一句话**：每轮找到最小值，和当前位置交换。
```java
public static void selectionSort(int[] arr) {
    int n = arr.length;
    for (int i = 0; i < n - 1; i++) {
        int minIndex = i; // 记录最小值下标
        // 找最小值
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j;
            }
        }
        // 交换
        int temp = arr[i];
        arr[i] = arr[minIndex];
        arr[minIndex] = temp;
    }
}
```

## 3. 插入排序（近乎有序时极快）
**一句话**：像整理扑克牌，把当前数插入前面有序序列。
```java
public static void insertionSort(int[] arr) {
    int n = arr.length;
    for (int i = 1; i < n; i++) {
        int current = arr[i]; // 要插入的数
        int j = i - 1;
        // 比 current 大的数后移
        while (j >= 0 && arr[j] > current) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = current; // 插入
    }
}
```

## 4. 希尔排序（插入排序升级版）
**一句话**：按步长分组插入排序，逐步缩小步长，最后变成插入排序。
```java
public static void shellSort(int[] arr) {
    int n = arr.length;
    // 步长 gap 每次减半
    for (int gap = n / 2; gap > 0; gap /= 2) {
        // 每组做插入排序
        for (int i = gap; i < n; i++) {
            int temp = arr[i];
            int j;
            for (j = i; j >= gap && arr[j - gap] > temp; j -= gap) {
                arr[j] = arr[j - gap];
            }
            arr[j] = temp;
        }
    }
}
```

## 5. 快速排序（面试必考！）
**一句话**：选一个基准数，小的放左边，大的放右边，递归左右。
```java
public static void quickSort(int[] arr) {
    quickSort(arr, 0, arr.length - 1);
}

private static void quickSort(int[] arr, int left, int right) {
    if (left >= right) return;
    int pivot = partition(arr, left, right); // 基准位置
    quickSort(arr, left, pivot - 1);  // 左递归
    quickSort(arr, pivot + 1, right); // 右递归
}

// 分区：把小的放左，大的放右
private static int partition(int[] arr, int left, int right) {
    int pivot = arr[left]; // 选第一个为基准
    int i = left, j = right;
    while (i < j) {
        // 右边找小的
        while (i < j && arr[j] >= pivot) j--;
        if (i < j) arr[i++] = arr[j];
        // 左边找大的
        while (i < j && arr[i] <= pivot) i++;
        if (i < j) arr[j--] = arr[i];
    }
    arr[i] = pivot; // 基准归位
    return i;
}
```

## 6. 归并排序（稳定且高效）
**一句话**：先拆分成最小单元，再两两合并成有序数组。
```java
public static void mergeSort(int[] arr) {
    if (arr == null || arr.length <= 1) return;
    int[] temp = new int[arr.length];
    mergeSort(arr, 0, arr.length - 1, temp);
}

private static void mergeSort(int[] arr, int left, int right, int[] temp) {
    if (left < right) {
        int mid = (left + right) / 2;
        mergeSort(arr, left, mid, temp);    // 左
        mergeSort(arr, mid + 1, right, temp); // 右
        merge(arr, left, mid, right, temp); // 合并
    }
}

// 合并两个有序数组
private static void merge(int[] arr, int left, int mid, int right, int[] temp) {
    int i = left, j = mid + 1, k = 0;
    while (i <= mid && j <= right) {
        if (arr[i] <= arr[j]) temp[k++] = arr[i++];
        else temp[k++] = arr[j++];
    }
    // 剩余元素拷贝
    while (i <= mid) temp[k++] = arr[i++];
    while (j <= right) temp[k++] = arr[j++];
    // 复制回原数组
    System.arraycopy(temp, 0, arr, left, k);
}
```

# 超级速记口诀
1. **冒泡**：两两交换，大的后移
2. **选择**：找最小，交换位置
3. **插入**：往前插，大的后移
4. **希尔**：分组插入，步长减半
5. **快排**：选基准，分左右，递归
6. **归并**：先拆分，再合并

---

### 总结
1. **简单排序**：冒泡、选择、插入，O(n²)，适合小数据量
2. **高效排序**：希尔、快排、归并，O(n log n)，适合大数据量
3. **稳定排序**：冒泡、插入、归并（相等元素不改变顺序）
4. **面试重点**：**快排、归并**必须能手写
